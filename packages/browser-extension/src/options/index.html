Think-Prompt for Web

Local-first prompt coach for your favourite LLM web apps. Everything stays on this device.

Sites

Privacy

No external network calls. Prompts are stored in IndexedDB on this device and optionally forwarded to your local Think-Prompt agent (http://127.0.0.1:47823).

PII (emails, phone numbers, Korean RRN, credit cards, API keys, JWTs, IP addresses) is masked on-device before anything is written.

Disable sites here, or uninstall the extension to remove it entirely. See full privacy policy.
